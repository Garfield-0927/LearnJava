# Chapter 5 Loop

### 5.1 while循环

语法：

```java
while(condition){
    // do something...
}
```

### 5.2 Do-While循环

语法：

```java
do{
    // do something
} while(condition)
```



### 5.3 For循环

语法：

```java
for(initial action; condition; iteration){
    // do something
}
```



### 5.4 增强For循环

语法：

```java
for(elementType val : arrayRef){
    // do something
}
```



### 5.5 Break和Continue

- break：结束当前循环
- continue： 跳过当前循环
